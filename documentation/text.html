Spark(string selector, element context).text(string text, boolean append)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • text - The text you wish to replace or append the current text with.
  • append - If true the new content will be appended to the current content.

Returns

If the text is not set then the text of the element will be returned. The text is content of the element without any tags.

Examples

Change the text of p tags with a class of 'change' to say 'Hello, World!'.

Spark('p.change').text('Hello, World!');

Append a full stop to the end of every p tag.

Spark('p').text('.', true);
