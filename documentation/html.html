Spark(string selector, element context).html(string html, boolean append)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • html - The html you wish to replace or append the elements inner html with.
  • append - If true the new content will be appended to the current content.

Returns

If the html is not set then the inner html of the element will be returned.

Examples

Change the html of p tags with a class of 'change' to say 'Hello, World!' in bold text.

Spark('p.change').html('<strong>Hello, World!</strong>');

Append an italic pipe character to every p tag.

Spark('p').text(' <em>|</em>', true);
