Spark.cookie(string name, string content, integer duration)

Parameters

  • name - The name of your cookie.
  • content - The content you wish to set.
  • duration - The duration of your cookie in milliseconds.

Returns

If the content is not set then the content of the cookie will be returned. If the cookie does not exist then it will return false.

Examples

Get the contents of a cookie.

Spark.cookie('myCookie');

Create a cookie that lasts until the user closes their browser.

Spark.cookie('myCookie', 'Hello, World!');

Create a cookie that lasts for 30 seconds, which happens to be 30000 milliseconds.

Spark.cookie('myCookie', 'Hello, World!', 30000);
