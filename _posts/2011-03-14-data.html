Spark.data(object element, string name, mixed data)

Parameters

  • element - The element you wish to assign data to.
  • name - The name you wish to assign your data to, so you can retrieve it.
  • data - The data to wish to assign, this can be anything.

Returns

If no data parameter is provided, it returns the content of the specified section.

Examples

Save a string to a clicked div tag.

Spark('div').event('click', function(e) {
    Spark.data(e, 'dataName', 'dataContent');
});

Display the stored data in a clicked div tag.

Spark('div').event('click', function(e) {
    alert(Spark.data(e, 'dataName'));
});
