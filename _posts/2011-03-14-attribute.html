Spark(string element, object context).attribute(object properties)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • properties - The properties you wish to assign.

Returns

Returns all of the properties of the element if the properties argument is not passed. It basically just returns the element object.

Examples

Set the title of all paragraph tags.

Spark('p.example').attribute({
    title: 'Hello, World!'
});

Get the title of a paragraph tag.

alert(Spark('p.example').attribute().title);
