Spark(string element, object context).css(object properties)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • properties - The CSS properties you wish to assign.

Returns

Returns all of the CSS properties of the element if no properties are supplied.

Examples

Set the colour of all paragraph tags to red.

Spark('p').css({
    color: '#FF0000'
});

Get the colour of a specific p tag

alert(Spark('p#postDescription').css().color);
