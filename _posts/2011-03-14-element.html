--- layout: default title: element description: Insert, prepend, append or remove an element from the specified elements. keywords: spark, documentation, element categories: documentation ---

Spark(string element, object context).element(string method, string tag name, object attributes, object styles, function callback)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • method - The action you want to take place, it can be one of the following.
    • insert
    • append
    • prepend
    • remove
  • tag name - The type of tag you wish to create, such as 'div' or 'p'.
  • attributes - An object of attributes to assign to the element such as 'innerHTML'.
  • styles - An object of style you wish to apply to the element such as 'color' or 'text-decoration'.
  • callback - The function you wish the array of inserted elements references to be passed to.

Returns

Nothing.

Examples

Add an empty p tag to every div.

Spark('div').element('insert', 'p');

Add a p tag with the content 'Hello, World!' to every div.

Spark('div').element('insert', 'p', {
    innerHTML: 'Hello, World!'
});

Add a p tag with content and an underline to every div.

Spark('div').element('insert', 'p', {
    innerHTML: 'Hello, World!'
}, {
    'text-decoration': 'underline'
});

Insert a p tag after every div.

Spark('div').element('append', 'p');

Insert a p tag before every div and log the tags that where added.

Spark('div').element('prepend', 'p', false, false, function(elements) {
    for(var i = 0; i < elements.length; i++) {
        console.log(elements[i])
    }
});

Remove every p tag.

Spark('p').element('remove');
