Spark.fixEvent(object event)

Parameters

  • event - The event to be made cross browser.

Returns

Returns the changed event object that has had its values altered to be cross browser, these include the pageX/Y problem with IE and the target/srcElement problem, also with IE. So you can use target and pageX/Y with no fear.

Examples

Make an event passed by a click tag cross browser.

function clickHandler(e) {
    e = Spark.fixEvent(e);
}
