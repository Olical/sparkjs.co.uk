Spark(string element, object context).classes(string method, string class)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • method - The method you wish the function to execute. Add, remove or has.
  • class - The class name you wish to add, remove or check for its existence.

Returns

If you use the has method then it returns true if it does have the class and false if it does not.

Examples

Check if an element has the selected class.

var hasClass = Spark('p#anId').classes('has', 'selected');
console.log(hasClass ? 'Yes' : 'No');

Add the class of selected to an element.

Spark('p#anId').classes('add', 'selected');

Remove the class of selected from an element.

Spark('p#anId').classes('remove', 'selected');
