Spark(string element, object context).each(function callback)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • callback - The function you wish to be run with the element passed to it.

Returns

Nothing.

Examples

Set the title of every p tag

Spark('p').each(function(e) {
    e.title = 'Set via the each function';
});
