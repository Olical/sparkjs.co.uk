Spark(string element, object context).computed()

Parameters

  • element - The element you wish to assign data to.
  • name - The name you wish to assign your data to, so you can retrieve it.

Returns

Returns an object containing all of the computed styles of the specified element.

Examples

Get the computed width of an element.

alert(Spark('p').computed().width);
