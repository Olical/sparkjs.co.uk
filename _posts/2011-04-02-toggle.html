--- layout: default title: toggle description: Show or hide an element depending if it is already visible with a specified transition. keywords: spark, documentation, toggle categories: documentation ---

Spark(string element, object context).toggle(string method, integer timeframe, string easing, function callback)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • method - The transition you wish to take place, please use one of the following.
    • slide
    • fade
    • sneak
    • visibility (the same as not passing it at all)
  • easing - The easing method to use, if false, it defaults to outQuad. All methods must be prepended with 'in', 'out' or 'inOut'.
    • Quad
    • Cubic
    • Quart
    • Quint
    • Sine
    • Expo
    • Circ
    • Elastic
    • Back
    • Bounce
  • timeframe - The amount of time you wish it to take place over.
  • callback - The function you wish to be run on completion.

Returns

Nothing.

Examples

Toggle the visibility of an element with no animation.

Spark('p.switch').toggle();

Toggle the visibility of an element with a fading animation.

Spark('p.switch').toggle('fade');

Toggle the visibility of an element with a sliding animation, default easing, default timeframe and a callback..

Spark('p.switch').toggle('slide', false, false, function() {
	console.log('Finished');
});
