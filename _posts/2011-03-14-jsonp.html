Spark.jsonp(string file, string callback, string parameters)

Parameters

  • file - The path to the file you wish to load.
  • callback - The name of the function you wish the data to be passed to.
  • parameters - Any other parameters you may need to pass to the file.

Returns

Nothing.

Examples

Load and display a tweet from @OliverCaldwell.

Spark.jsonp('http://search.twitter.com/search.json', 'loaded', 'q=from%3AOliverCaldwell');

window.loaded = function(data) {
    Spark('p.target').html(data.results[0].text);
}
