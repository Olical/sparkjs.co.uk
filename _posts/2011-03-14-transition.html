Spark(string element, object context).transition(string method, integer timeframe, string easing, function callback)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • method - The transition you wish to take place, please use one of the following.
    • slidedown
    • slideup
    • fadein
    • fadeout
    • sneakin
    • sneakout
  • easing - The easing method to use, if false, it defaults to outQuad. All methods must be prepended with 'in', 'out' or 'inOut'.
    • Quad
    • Cubic
    • Quart
    • Quint
    • Sine
    • Expo
    • Circ
    • Elastic
    • Back
    • Bounce
  • timeframe - The amount of time you wish it to take place over.
  • callback - The function you wish to be run on completion.

Returns

Nothing.

Examples

Fade an element out then slide it down with a bouncing effect.

Spark('div.alert').transition('fadeout', 400, false, function() {
    Spark('div.alert').transition('slidedown', 500, 'outBounce');
});
