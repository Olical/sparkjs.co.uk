Spark.client()

Parameters

None.

Returns

Returns an object containing the following values.

  • browser - The name of the browser.
  • version - The version of the browser.
  • os - The operating system that the browser is running on.

Examples

Get all of the information on the client and display it.

var client = Spark.client();

Spark('p.browser').html(
    'You are using ' + client.browser +
    ' version ' + client.version +
    ' on ' + client.os
);
