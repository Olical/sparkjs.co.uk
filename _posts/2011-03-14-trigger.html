Spark(string element, object context).trigger(string type)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • type - The type of event you wish to trigger, it can be one of the following
    • load
    • unload
    • blur
    • change
    • focus
    • reset
    • select
    • submit
    • abort
    • keydown
    • keypress
    • keyup
    • click
    • dblclick
    • mousedown
    • mousemove
    • mouseout
    • mouseover
    • mouseup

Returns

Nothing.

Examples

Set a click event on an element and then trigger it.

Spark('div.button').event('click', function(e) {
    alert('Button clicked.');
});

Spark('div.button').trigger('click');
