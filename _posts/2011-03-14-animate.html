Spark(string element, object context).animate(object properties, integer timeframe, string easing, function callback)

Parameters

  • element - The CSS selector string to your element.
  • context - If supplied, the context in which you wish the selection to take place.
  • properties - The properties you wish to animate and the value you wish it to animate to.
  • timeframe - The amount of milliseconds you wish the animation to take.
  • easing - The easing method to use, if false, it defaults to outQuad. All methods must be prepended with 'in', 'out' or 'inOut'.
    • Quad
    • Cubic
    • Quart
    • Quint
    • Sine
    • Expo
    • Circ
    • Elastic
    • Back
    • Bounce
  • callback - The function you wish to be run on completion.

Returns

Nothing.

Examples

Animate elements opacity to 0.5 over two seconds and alert when done with an elastic easing.

Spark('p').animate({opacity: 0.5}, 2000, 'inElastic', function() {
    alert('Done');
});

Animate elements to 20 pixels in width with the default timeframe and easing. When done run a callback.

Spark('div').animate({width: 20}, false, false, function() {
    alert('Done');
});
