Each documentation page contains the anatomy, parameters, return values and examples of the specified function.

Most functions can be chained, the only ones that can't are functions such as ajax which do not involve elements. When chaining animations they are run one after the other, like so.

Spark('div.something')
    .animate({width: 50})
    .animate({opacity: 0.5});

You can also add your own functions via plugins. Here is a plugin template for you to copy and adapt.

SparkFn.pluginName = function(argument) {
    // Set up any variables
    var element = null,
        e = null;
    
    // Loop through all of the elements
    for(e in this.elements) {
        // Grab the current element
        element = this.elements[e];
        
        // Alter the element variable
    }
    
    // Return the Spark object to allow chaining
    return this;
};

// Reinitialise Spark
SparkIn();
